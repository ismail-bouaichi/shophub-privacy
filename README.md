# ShopHub Privacy Policy

This repository hosts the privacy policy for the ShopHub mobile application.

## About ShopHub

ShopHub is a demo/portfolio e-commerce mobile application built with Flutter. The app features:
- Product browsing and shopping cart functionality
- Google Maps integration for location-based services
- Local data storage using secure storage mechanisms
- Address selection and delivery tracking features

## Privacy Policy

The privacy policy is available in HTML format and covers:
- Information collection practices
- Data usage and storage
- Third-party service integrations (Google Maps)
- User rights and data protection
- GDPR and COPPA compliance

## Viewing the Privacy Policy

You can view the privacy policy by opening [shophub-privacy.html](shophub-privacy.html) in any web browser.

**Live URL:** [https://ismail-bouaichi.github.io/shophub-privacy/shophub-privacy.html](https://ismail-bouaichi.github.io/shophub-privacy/shophub-privacy.html)

## Key Points

- **No External Data Transmission:** All user data is stored locally on the device
- **Location Services:** Used only for Google Maps integration with user permission
- **Demo Application:** This is a portfolio project with static data
- **Data Deletion:** Users can delete all data by uninstalling the app
